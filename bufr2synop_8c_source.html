20 /*!
21  \file bufr2synop.c
22  \brief decodes a bufr file using ECMWF bufr library and tries to pass the decoded reports to synop-format
23 
24  \mainpage Bufr to alphanumeric
25 
26  \section Introduction
27  This is a package to make the transition to bufr reports from alphanumeric text easiest as possible.
28 
29  A lot of software is coded assuming that the primary source of meteorological reports is in alphanumeric format. Decode libraries are expecting this. But time is changing, meteorological services are migrating to bufr and other binary formats. Most decode sofware have to be changed.
30 
31  This is a software to get meteorological reports in old alphanumeric format from bufr files. At the moment includes the following reports:
32 
33  - FM 12-XIV SYNOP
34  - FM 13-XIV SHIP
35  - FM 14-XIV SYNOP MOBIL
36  - FM 18-XII BUOY
37 
38  The software is based in bufrdc library from ECMWF It must to be installed. In that package there is a \a synop2bufr application doing just the reverse task.
39 
40  Note that the results from this library is not intended to match at %100 level to original alphanumeric reports. It cannot. Some variables in alphanumeric code rules can be coded in several ways, and there is not a regional even national decision about them. As example, the 'hshs' item (table code 1617) for synop FM 12 can be coded using 00-80 range or 90-99 one. A numeric value for heigh of base clouds can be coded in two ways. And there some few more examples.
41 
42  \section Install
43 
44  Remember you have to download and install bufrdc library from ECMWF
45 
46  http://old.ecmwf.int/products/data/software/bufr.html
47 
48  You also will need a Fortran and C compilers in your system and the usual development packages. GNU gcc and gfortran compilers are ok.
49 
50  Then, to install bufr2synop, download the tarball and uncompress it
51 
52  \code
53  tar -xvzf bufr2synop-X.Y.tar.gz
54  \endcode
55 
56  Where X.Y is the current version. This will create the directory \a bufr2synop-X.Y
57 
58  You then have to configure
59 
60  \code
61  cd bufr2synop-X.Y
62  ./configure
63  \endcode
64 
65  after a succesfully configuration you then can then compile the package
66 
67  \code
68  make
69  \endcode
70 
71  and finally install it.
72  \code
73  make install
74  \endcode
75 
76  by default, it will install binaries and libraries under \a /usr/local tree, so you will need root privileges.
77 
78 */
